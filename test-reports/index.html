Architecture Implementation

JWT Authentication

• 15-minute token expiry for security balance
• Argon2id password hashing (OWASP recommended)
• Bearer token format in Authorization header
• Claims-based identity with UserId, Email, Roles

Role-Based Access Control

• SuperAdmin: All permissions, can manage users/roles
• DivisionAdmin: Can manage assigned division(s) and all teams within
• TeamAdmin: Can manage only assigned team(s) and their players
• Hierarchical authorization with resource ownership checks

Audit Trail

• All entity changes tracked with CreatedByUserId / UpdatedByUserId
• Old and new values stored as JSON for change history
• SuperAdmin-only audit log API with filtering capabilities
• Serilog structured logging for production debugging
