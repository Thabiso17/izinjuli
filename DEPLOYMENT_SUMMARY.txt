STEP 1: Deploy Code to Railway
   $ git push origin main
   → Railway auto-deploys on push
   → Wait for build to complete (check dashboard)

STEP 2: Run Database Migration
   $ npm install -g @railway/cli
   $ railway login
   $ railway link
   $ railway run dotnet ef database update \
     --project iDiski.Infrastructure \
     --startup-project iDiski.Api
   → Expected: "Applying migration 'AddAuthenticationSystem'. Done."

STEP 3: Set Environment Variables
   In Railway Dashboard → iDiski API Service → Variables:
   
   Jwt__SecretKey = [RUN: openssl rand -base64 64]
   Jwt__Issuer = https://api.izinjuli.vercel.app
   Jwt__Audience = https://izinjuli.vercel.app
   
   → Click "Redeploy" on API service
